CONSEJOS PARA PREPARAR LAS IMÁGENES:
=====================================

1. Formato recomendado: PNG con transparencia

2. Optimización:
   - Comprime las imágenes para web (puedes usar TinyPNG.com)
   - No necesitas resoluciones muy altas (72-96 DPI está bien)

3. Nombres de archivo:
   - IMPORTANTE: Los nombres deben ser EXACTAMENTE como se indican arriba
   - Respeta mayúsculas y minúsculas
   - No uses espacios ni caracteres especiales

4. Colores:
   - Los iconos sociales pueden ser blancos si el banner es verde (#84ac74)
   - Alternativamente, pueden ser de color si tienen fondo transparente

5. Si no tienes alguna imagen:
   - El sistema ocultará automáticamente las imágenes que falten
   - No se romperá la página, pero se verá incompleta

VERIFICACIÓN:
=============

Para verificar que las imágenes están correctamente colocadas:

1. Abre tu navegador
2. Ve a: http://localhost/OJS/amcad-portal/index-simple.php
3. Presiona F12 para abrir las herramientas de desarrollador
4. Ve a la pestaña "Console"
5. Si hay imágenes faltantes, verás errores 404

Alternativamente, abre esta carpeta en el explorador de Windows y
verifica que tienes exactamente 7 archivos PNG.
